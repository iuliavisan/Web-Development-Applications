# Lab 3: Advanced CSS Selectors and External Stylesheets

## EN
**Project Overview**
This laboratory focuses on optimizing web development workflows by moving away from inline styling towards centralized CSS management. The exercises demonstrate how to apply consistent styles across multiple elements using classes, IDs, and external style sheets.

**Directory Content**
Based on the laboratory exercises and folder structure, this directory contains:
* P1.html: Initial exercise demonstrating the inefficiency of styling each HTML element individually and the transition to header-defined styles.
* P2.html: Implementation of internal CSS within the <style> tag to define global rules for paragraphs (Arial, red color, bold) and headings.
* P3.html: Introduction to External Stylesheets, where styles are moved to a separate .css file for better code maintainability.
* P4.html: Use of CSS Classes to apply specific styles (e.g., "stilimpar") to alternating elements or specific groups of tags.
* P5.html: Advanced styling using ID selectors (#) for unique elements and applying multiple classes to a single HTML tag for modular design.
* stil.css: The external stylesheet containing all visual rules, linked across different HTML pages to ensure a unified design.

**Technical Skills Demonstrated**
* Code Efficiency: Transitioning from element-level styling to global CSS rules to reduce redundancy.
* External Styling: Linking external .css files to HTML documents for professional project organization.
* CSS Classes & IDs: Using classes for reusable styles and IDs for unique page elements.
* Typography & Color Control: Precise management of font families, sizes, weights, and text colors across complex documents.

## RO
**Prezentare Proiect**
Acest laborator se concentrează pe optimizarea fluxului de lucru în dezvoltarea web, trecând de la stilizarea individuală a elementelor la gestionarea centralizată prin CSS. Exercițiile demonstrează cum se pot aplica stiluri consistente pe mai multe elemente folosind clase, ID-uri și foi de stil externe.

**Conținutul Directorului**
Conform exercițiilor din laborator și structurii folderului, acest director conține:
* P1.html: Exercițiu inițial care demonstrează ineficiența stilizării individuale a fiecărui element și tranziția către stiluri definite în zona de header.
* P2.html: Implementarea CSS-ului intern în cadrul tag-ului <style> pentru a defini reguli globale pentru paragrafe (Arial, culoare roșie, bold) și titluri.
* P3.html: Introducerea Foilor de Stil Externe, unde regulile de design sunt mutate într-un fișier .css separat pentru o mai bună mentenanță a codului.
* P4.html: Utilizarea claselor CSS pentru a aplica stiluri specifice (ex: "stilimpar") pe elemente alternative sau grupuri specifice de tag-uri.
* P5.html: Stilizare avansată folosind selectori de tip ID (#) pentru elemente unice și aplicarea mai multor clase pe un singur tag HTML pentru un design modular.
* stil.css: Foaia de stil externă care conține toate regulile vizuale, conectată la diferite pagini HTML pentru a asigura un design unitar.

**Competențe Tehnice Demonstrate**
* Eficientizarea Codului: Tranziția de la stilizarea la nivel de element la reguli CSS globale pentru a reduce redundanța.
* Stilizare Externă: Conectarea fișierelor .css externe la documentele HTML pentru o organizare profesională a proiectului.
* Clase și ID-uri CSS: Utilizarea claselor pentru stiluri reutilizabile și a ID-urilor pentru elemente unice în pagină.
* Controlul Tipografiei și Culorilor: Gestionarea precisă a fonturilor, dimensiunilor și culorilor textului în documente complexe.
