# Cell 03: Introduction to JavaScript and Dynamic Interactions

## EN
**Project Overview**
This cell marks the introduction of JavaScript as the primary tool for creating dynamic and interactive web experiences. The exercises focus on event handling, DOM manipulation through scripts, and the implementation of interactive UI components that respond to user actions in real-time.

**Directory Content**
Based on the exercise requirements (Cell 3-0 to 3-4), this directory contains:
* **ex00 (Background Color Change)**: A web page featuring a button that, when clicked, changes the background to a random color. This demonstrates basic event listening and programmatic CSS manipulation.
* **ex01 (Balloon)**: Implementation of a 200px red circle (balloon) that grows by 10px on click and cycles through colors (Red -> Green -> Blue). When the mouse leaves the balloon, it shrinks by 5px and cycles colors in reverse.
* **ex02 (Calc)**: A functional web calculator that performs basic arithmetic operations (addition, subtraction, multiplication, division, modulo). It includes validation to prevent division by zero (displaying "It's over 9000!") and logs results to both an alert and the console every 30 seconds.
* **ex03 (Todo List)**: A dynamic To-Do list application where users can add new tasks via a prompt. Tasks are saved in the browser's cookies to persist after a page refresh. Clicking a task allows the user to delete it after a confirmation prompt.
* **ex04 (Ex02 & Ex03 with jQuery)**: A refactored version of the Calculator and To-Do list using the jQuery library, demonstrating how to achieve the same results with a more concise and cross-browser compatible syntax.

**Technical Skills Demonstrated**
* **JavaScript Event Handling**: Using `onclick`, `onmouseenter`, and `onmouseleave` to trigger logic.
* **DOM Manipulation**: Dynamically creating elements, modifying styles, and updating content through scripts.
* **Data Persistence**: Implementing browser cookies to save user data locally.
* **Input Validation**: Ensuring mathematical logic is sound and handling edge cases like division by zero.
* **Library Integration**: Utilizing jQuery for more efficient DOM selection and event management.

## RO
**Prezentare Proiect**
Această celulă marchează introducerea JavaScript ca instrument principal pentru crearea experiențelor web dinamice și interactive. Exercițiile se concentrează pe gestionarea evenimentelor, manipularea DOM-ului prin scripturi și implementarea componentelor UI interactive care răspund la acțiunile utilizatorului în timp real.

**Conținutul Directorului**
Conform cerințelor exercițiilor (Cell 3-0 până la 3-4), acest director conține:
* **ex00 (Schimbarea culorii de fundal)**: O pagină web cu un buton care, la click, schimbă fundalul într-o culoare aleatorie. Aceasta demonstrează ascultarea evenimentelor și manipularea programatică a CSS-ului.
* **ex01 (Balonul)**: Implementarea unui cerc roșu de 200px (balon) care crește cu 10px la click și schimbă culorile (Roșu -> Verde -> Albastru). Când mouse-ul părăsește balonul, acesta se micșorează cu 5px și schimbă culorile în ordine inversă.
* **ex02 (Calculator)**: Un calculator web funcțional care efectuează operații aritmetice de bază. Include validare pentru a preveni împărțirea la zero (afișând "It's over 9000!") și afișează rezultatele atât într-o alertă, cât și în consolă la fiecare 30 de secunde.
* **ex03 (Listă de sarcini - Todo List)**: O aplicație dinamică de tip To-Do list unde utilizatorii pot adăuga sarcini noi printr-un prompt. Sarcinile sunt salvate în cookie-urile browserului pentru a persista după reîmprospătarea paginii.
* **ex04 (Ex02 & Ex03 cu jQuery)**: O versiune refăcută a Calculatorului și a Listei de sarcini folosind biblioteca jQuery, demonstrând cum se pot obține aceleași rezultate cu o sintaxă mai concisă.

**Competențe Tehnice Demonstrate**
* **Gestionarea Evenimentelor JavaScript**: Utilizarea `onclick`, `onmouseenter` și `onmouseleave` pentru a declanșa logica.
* **Manipularea DOM**: Crearea dinamică a elementelor, modificarea stilurilor și actualizarea conținutului prin scripturi.
* **Persistența Datelor**: Implementarea cookie-urilor de browser pentru a salva datele utilizatorului local.
* **Validarea Datelor de Intrare**: Asigurarea logicii matematice corecte și gestionarea cazurilor speciale (împărțirea la zero).
* **Integrarea Bibliotecilor Externe**: Utilizarea jQuery pentru selectarea mai eficientă a elementelor DOM și gestionarea evenimentelor.
