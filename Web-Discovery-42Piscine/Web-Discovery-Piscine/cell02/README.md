# Cell 02: Advanced CSS Layouts and Profile Design

## EN
**Project Overview**
This cell focuses on advanced CSS techniques to create complex visual effects and structured layouts. The exercises introduce the separation of concerns by utilizing external stylesheets (.css) and explore specialized web design patterns like dropdown menus, parallax scrolling, and personal branding cards.

**Directory Content**
Based on the exercise requirements (Cell 2-0 to 2-3), this directory contains:
* **ex00 (Improved responsive)**: A refined version of the previous table layout, now using a dedicated `responsive.css` file to ensure the content covers the entire page and scales smoothly.
* **ex01 (SNCF Sandwich)**: Implementation of a ham-and-cheese style dropdown menu using only HTML and CSS (no JavaScript allowed). The focus is on creating a functional and appetizing navigation interface.
* **ex02 (Parallax)**: A web page demonstrating the parallax scrolling effect. It features at least two images that stay in the background while text blocks and other content move over them at different speeds.
* **ex03 (User profile card)**: Creation of a professional profile card using an invisible table for alignment. It includes a profile picture, login details, social network icons, and a "Contact" button.



**Technical Skills Demonstrated**
* **File Separation**: Transitioning to professional workflows by splitting HTML structure and CSS presentation into separate files.
* **CSS Interactive Menus**: Using pseudo-classes like `:hover` to create functional dropdown navigation without scripts.
* **Visual Storytelling**: Implementing Parallax effects to create depth and an engaging user experience during scrolling.
* **UI/UX Component Design**: Building a structured User Profile Card with specific alignment, typography, and iconography constraints.

## RO
**Prezentare Proiect**
Această celulă se concentrează pe tehnici avansate CSS pentru a crea efecte vizuale complexe și layout-uri structurate. Exercițiile introduc separarea conceptelor prin utilizarea foilor de stil externe (.css) și explorează modele de design web specializate, cum ar fi meniurile de tip dropdown, scrolling-ul de tip parallax și cardurile de branding personal.

**Conținutul Directorului**
Conform cerințelor exercițiilor (Cell 2-0 până la 2-3), acest director conține:
* **ex00 (Improved responsive)**: O versiune rafinată a layout-ului tabelar anterior, folosind acum un fișier dedicat `responsive.css` pentru a asigura acoperirea întregii pagini și scalarea fluidă.
* **ex01 (SNCF Sandwich)**: Implementarea unui meniu de tip dropdown folosind exclusiv HTML și CSS (fără JavaScript). Accentul este pus pe crearea unei interfețe de navigare funcționale și estetice.
* **ex02 (Parallax)**: O pagină web care demonstrează efectul de scrolling parallax. Include cel puțin două imagini care rămân în fundal în timp ce blocurile de text și restul conținutului se mișcă peste ele la viteze diferite.
* **ex03 (User profile card)**: Crearea unui card de profil profesional folosind un tabel invizibil pentru aliniere. Include o poză de profil, detalii de login, pictograme pentru rețele sociale și un buton de "Contact".

**Competențe Tehnice Demonstrate**
* **Separarea Fișierelor**: Tranziția către fluxuri de lucru profesionale prin separarea structurii HTML de prezentarea CSS în fișiere diferite.
* **Meniuri Interactive CSS**: Utilizarea pseudo-claselor precum `:hover` pentru a crea navigare de tip dropdown funcțională fără scripturi.
* **Storytelling Vizual**: Implementarea efectelor de tip Parallax pentru a crea profunzime și o experiență de utilizare captivantă în timpul scrolling-ului.
* **Design de Componente UI/UX**: Construirea unui card de profil structurat, respectând constrângeri specifice de aliniere, tipografie și iconografie.
