# Lab 5: Interactive Components and Responsive Restaurant Layout

## EN
**Project Overview**
This laboratory focuses on building semi-complex web interfaces by combining structural HTML5 semantic elements with advanced external CSS styling. The exercises simulate real-world scenarios, such as creating a business website (Restaurant) and integrated contact systems.

**Directory Content**
Based on the laboratory exercises and folder structure, this directory contains:
* Problema1.html: A dedicated page featuring a horizontal navigation bar (nav, ul, li) and a styled contact form with name, email, and message fields.
* Problema1.css: External stylesheet for the first exercise, managing the navigation hover effects and form spacing.
* Problema2.html: A complete Restaurant Landing Page structure. It includes a header with a navigation menu (Menu, About, Reservations), a descriptive "About" section with images, a responsive Menu grid, and a reservation form.
* Problema2.css: Comprehensive external stylesheet implementing a responsive layout, Flexbox/Grid for the menu, and transparency effects for the header.
* restaurant.jpg & restaurant2.jpg: Visual assets used to populate the "About" and "Menu" sections of the restaurant project.

**Technical Skills Demonstrated**
* Professional Navigation: Building horizontal navigation bars with semantic HTML5 tags and interactive hover states.
* Responsive Design: Implementing layouts that adapt to different screen sizes using margins, padding, and flexible containers.
* Advanced Form Styling: Creating attractive user interfaces for data collection (Contact and Reservations) with customized buttons and input fields.
* Component-Based Styling: Using external CSS to separate logic from presentation, ensuring consistency across sections like "About", "Menu", and "Footer".

## RO
**Prezentare Proiect**
Acest laborator se concentrează pe construirea unor interfețe web semi-complexe prin combinarea elementelor semantice HTML5 cu stilizarea avansată prin CSS extern. Exercițiile simulează scenarii reale, cum ar fi crearea unui site de prezentare pentru o afacere (Restaurant) și sisteme integrate de contact.

**Conținutul Directorului**
Conform exercițiilor din laborator și structurii folderului, acest director conține:
* Problema1.html: O pagină dedicată care include o bară de navigație orizontală (nav, ul, li) și un formular de contact stilizat cu câmpuri pentru nume, email și mesaj.
* Problema1.css: Foaia de stil externă pentru primul exercițiu, care gestionează efectele de hover ale navigării și spațierea formularului.
* Problema2.html: O structură completă pentru o pagină de restaurant (Landing Page). Include un antet cu meniu de navigație (Meniu, Despre, Rezervări), o secțiune descriptivă „Despre” cu imagini, o grilă de meniu responsivă și un formular de rezervări.
* Problema2.css: Foaie de stil externă complexă care implementează un layout responsiv, Flexbox/Grid pentru meniu și efecte de transparență pentru antet.
* restaurant.jpg & restaurant2.jpg: Resurse vizuale utilizate pentru secțiunile „Despre” și „Meniu” ale proiectului restaurantului.

**Competențe Tehnice Demonstrate** * Navigare Profesională: Construirea 
barelor de navigație orizontale cu tag-uri semantice HTML5 și stări 
interactive de tip hover. * Design Responsiv: Implementarea 
layout-urilor care se adaptează la diferite dimensiuni de ecran folosind 
margini, padding și containere flexibile. * Stilizare Avansată a 
Formularelor: Crearea interfețelor atractive pentru colectarea datelor 
(Contact și Rezervări) cu butoane și câmpuri de intrare personalizate. * 
Stilizare bazată pe Componente: Utilizarea CSS-ului extern pentru a 
separa logica de prezentare, asigurând consistența între secțiuni precum 
„Despre”, „Meniu” și „Footer”.
