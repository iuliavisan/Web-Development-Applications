# Frontend Development University Course Portfolio

## EN
**Project Description**
This repository serves as a comprehensive portfolio of my work during the Frontend Development university course. It documents my progression from core web structural principles to advanced interactive styling and client-side scripting. Each module represents a specific milestone in mastering modern web standards.

**Curriculum Overview**
The course is structured into seven major laboratories, each focusing on key frontend competencies:
* **Lab 01 - HTML Layouts and Forms**: Mastering semantic structure and complex data collection interfaces.
* **Lab 02 - HTML5 Multimedia & CSS3**: Integrating native audio/video and exploring initial visual styling.
* **Lab 03 - Advanced CSS Selectors**: Implementing modular design patterns through external stylesheets and specific selectors.
* **Lab 04 - Box Model & Backgrounds**: Deep dive into the CSS Box Model, positioning, and layered background management.
* **Lab 05 - Interactive Components**: Building full-scale responsive landing pages with advanced UI elements.
* **Lab 06 - JavaScript & DOM Manipulation**: Transitioning to dynamic pages by manipulating the HTML structure via scripts.
* **Lab 07 - Interactive Feedback System**: Developing complex, state-managed widgets using integrated HTML, CSS, and JS logic.

**Technical Stack**
* **Languages**: HTML5, CSS3, JavaScript (ES6+).
* **Concepts**: Responsive Web Design (RWD), DOM API, Semantic Web, UI/UX Fundamentals.
* **Tools**: VS Code, Git/GitHub, XAMPP (for local environment testing).

## RO
**Descrierea Proiectului**
Acest repository reprezintă un portofoliu complet al activității mele în cadrul cursului universitar de Dezvoltare Frontend. Documentează progresul meu de la principiile de bază ale structurii web până la stilizarea interactivă avansată și scripting pe partea de client. Fiecare modul reprezintă o etapă specifică în stăpânirea standardelor web moderne.

**Structura Cursului**
Cursul este structurat în șapte laboratoare principale, fiecare concentrându-se pe competențe cheie de frontend:
* **Lab 01 - Layout-uri și Formulare HTML**: Stăpânirea structurii semantice și a interfețelor complexe de colectare a datelor.
* **Lab 02 - Multimedia HTML5 și CSS3**: Integrarea nativă a elementelor audio/video și explorarea stilizării vizuale inițiale.
* **Lab 03 - Selectori CSS Avansați**: Implementarea modelelor de design modular prin foi de stil externe și selectori specifici.
* **Lab 04 - Modelul Box și Fundaluri**: Analiză aprofundată a Modelului Box CSS, a poziționării și a gestionării fundalurilor stratificate.
* **Lab 05 - Componente Interactive**: Construirea paginilor de destinație responsive cu elemente avansate de interfață (UI).
* **Lab 06 - JavaScript și Manipularea DOM**: Tranziția către pagini dinamice prin manipularea structurii HTML via scripturi.
* **Lab 07 - Sistem Interactiv de Feedback**: Dezvoltarea unor widget-uri complexe cu gestionare de stare, folosind logică integrată HTML, CSS și JS.

**Tehnologii Utilizate**
* **Limbaje**: HTML5, CSS3, JavaScript (ES6+).
* **Concepte**: Design Web Responsiv (RWD), DOM API, Web Semantic, Fundamente UI/UX.
* **Instrumente**: VS Code, Git/GitHub, XAMPP (pentru testarea mediului local).
