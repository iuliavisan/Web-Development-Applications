# Lab 1: Fundamentals of HTML Structure, Layouts, and Forms

## EN
**Project Overview**
This first laboratory focuses on the fundamental syntax of HTML and the practical implementation of core web elements. It covers basic document structure, complex table layouts, multimedia integration, text formatting, and interactive web forms.

**Directory Content**
Based on the laboratory exercises and folder structure, this directory contains:
* index.html: Implements a structured page layout using a table border="1" with dedicated zones for the header, navigation menu, main content, and footer.
* pagina.html: Demonstrates basic page creation, image insertion (400x300px) with alternative text, and the use of internal bookmarks (anchors) to navigate within the same document.
* formatari.html: Showcases a wide range of HTML text formatting tags including font faces, bold, italic, subscript, superscript, mark, and various heading levels from h1 to h6.
* containere.html: Illustrates the difference between block-level (DIV) and inline (SPAN) containers used for logical document sectioning and specific text styling.
* formular.html: A comprehensive hotel reservation form utilizing fieldsets and legends. It includes text inputs for personal data, password fields, textareas for addresses, dropdown selects for room types, radio buttons for meal plans, and checkboxes for extra facilities.
* Fisiere/: A dedicated directory storing project assets such as header images, project photos, and downloadable documents like cv.pdf or cv.doc.

**Technical Skills Demonstrated**
* Structural Organization: Creating page layouts with table-based designs including colspan and rowspan attributes.
* Navigation Systems: Implementing absolute links to external sites (Google), relative links between local pages, and internal document bookmarks.
* Multimedia Handling: Embedding images using the <img> tag and external content using <iframe> to display websites within the page.
* Form Development: Building complex user interfaces for data collection, including file upload inputs for ID documents.

## RO
**Prezentare Proiect**
Acest prim laborator se concentrează pe sintaxa fundamentală a limbajului HTML și implementarea practică a elementelor web de bază. Acesta acoperă structura documentului, layout-uri complexe bazate pe tabele, integrarea elementelor multimedia, formatarea textului și crearea formularelor web interactive.

**Conținutul Directorului**
Conform exercițiilor din laborator și structurii folderului, acest director conține:
* index.html: Implementează un layout structurat folosind un tabel (table border="1") cu zone dedicate pentru antet, meniu de navigare, zona de conținut și subsol.
* pagina.html: Demonstrează crearea unei pagini de bază, inserarea imaginilor (400x300px) cu text alternativ și utilizarea semnelor de carte interne (ancore) pentru navigarea în cadrul aceluiași document.
* formatari.html: Prezintă o gamă largă de tag-uri pentru formatarea textului, inclusiv font face, bold, italic, indici (sub), exponenți (sup), marcaje de evidențiere și diverse niveluri de titluri de la h1 la h6.
* containere.html: Ilustrează diferența dintre containerele de tip "block" (DIV) și cele "inline" (SPAN) utilizate pentru secționarea logică a documentului și stilizarea specifică a textului.
* formular.html: Un formular complex pentru rezervare hotelieră, utilizând fieldset-uri și legende. Include câmpuri de text pentru date personale, parole, zone de text (textarea), liste de selecție pentru tipul camerei, butoane radio pentru opțiuni de masă și căsuțe de bifat pentru facilități extra.
* Fisiere/: Un director dedicat pentru stocarea resurselor precum imagini de antet, fotografii de proiect și documente descărcabile precum cv.pdf sau cv.doc.

**Competențe Tehnice Demonstrate**
* Organizare Structurală: Crearea layout-urilor de pagină prin design bazat pe structuri tabelare, inclusiv utilizarea atributelor colspan și rowspan.
* Sisteme de Navigare: Implementarea legăturilor absolute către site-uri externe (Google), a celor relative între paginile locale și a semnelor de carte interne.
* Gestionare Multimedia: Inserarea imaginilor prin tag-ul <img> și a conținutului extern prin <iframe> pentru afișarea site-urilor în interiorul paginii.
* Dezvoltare Formulare: Construirea interfețelor complexe pentru colectarea datelor, inclusiv câmpuri de tip "file" pentru încărcarea documentelor de identitate.
