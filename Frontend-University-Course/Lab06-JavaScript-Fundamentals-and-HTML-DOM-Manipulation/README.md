# Lab 6: JavaScript Fundamentals and HTML DOM Manipulation

## EN
**Project Overview**
This laboratory focuses on implementing client-side interactivity using JavaScript and the Document Object Model (DOM). The exercises cover script integration methods, dynamic content updates, attribute manipulation, and real-time CSS styling.

**Directory Content**
Based on the full laboratory documentation, this directory is organized into the following exercise modules:
* Problema 1: Demonstrates JavaScript integration methods including external files (*.js), header scripts, and body scripts to trigger alert messages.
* Problema 2: Interactive content updates using events (onclick, onmouseover) to replace existing text in <h1> and <div> elements with current date information.
* Problema 3: Dynamic attribute and content modification, where a single function changes a link's text, destination URL (href), and target property upon a button click.
* Problema 4: Demonstrates tag-based element access using getElementsByTagName() to extract and display content from specific indices within an array of elements.
* Problema 5: Implementation of a functional image gallery with "Previous" and "Next" navigation, utilizing ID-based selection to swap image sources and hyperlink targets.
* Problema 6: Real-time visual styling through the DOM, specifically changing the color property of heading elements via the style object.
* Problema 7: Advanced UI control demonstrating the difference between the visibility and display CSS properties to show or hide text and layout containers.
* fisier.js: External script file used for centralized logic and multi-page reusability.



**Technical Skills Demonstrated**
* DOM Navigation: Locating elements via getElementById() and managing collections with getElementsByTagName().
* Event-Driven Programming: Handling user interactions such as clicks, hover states, and double-clicks.
* Dynamic Content and Styling: Manipulating innerHTML, HTML attributes, and CSS styles directly through JavaScript.
* Modular Scripting: Separating logic into external files and functions for cleaner code architecture.

## RO
**Prezentare Proiect**
Acest laborator se concentrează pe implementarea interactivității pe partea de client folosind JavaScript și Document Object Model (DOM). Exercițiile acoperă metodele de integrare a scripturilor, actualizările dinamice de conținut, manipularea atributelor și stilizarea CSS în timp real.

**Conținutul Directorului**
Conform documentației complete de laborator, acest director este organizat în următoarele module de exerciții:
* Problema 1: Demonstrează metodele de integrare JavaScript, inclusiv fișiere externe (*.js), scripturi în header și în body pentru a declanșa mesaje de tip alertă.
* Problema 2: Actualizări interactive de conținut folosind evenimente (onclick, onmouseover) pentru a înlocui textul existent în elementele <h1> și <div> cu data curentă.
* Problema 3: Modificarea dinamică a atributelor și a conținutului, unde o singură funcție schimbă textul unui link, adresa URL de destinație (href) și proprietatea target la click-ul unui buton.
* Problema 4: Demonstrează accesarea elementelor după tag folosind getElementsByTagName() pentru a extrage și afișa conținutul de la indici specifici dintr-un array de elemente.
* Problema 5: Implementarea unei galerii de imagini funcționale cu navigare "Precedenta" și "Următoarea", utilizând selecția pe bază de ID pentru a schimba sursele imaginilor și link-urile atașate.
* Problema 6: Stilizarea vizuală în timp real prin DOM, în special schimbarea proprietății de culoare a elementelor de titlu prin obiectul style.
* Problema 7: Control avansat al interfeței demonstrând diferența dintre proprietățile CSS visibility și display pentru a ascunde sau afișa textul și containerele de layout.
* fisier.js: Fișier script extern utilizat pentru logică centralizată și reutilizare în mai multe pagini.



**Competențe Tehnice Demonstrate**
* Navigare DOM: Localizarea elementelor prin getElementById() și gestionarea colecțiilor cu getElementsByTagName().
* Programare bazată pe Evenimente: Gestionarea interacțiunilor utilizatorului, cum ar fi click-urile, stările de hover și dublu-click-urile.
* Conținut și Stilizare Dinamică: Manipularea innerHTML, a atributelor HTML și a stilurilor CSS direct prin JavaScript.
* Scripting Modular: Separarea logicii în fișiere externe și funcții pentru o arhitectură curată a codului.
