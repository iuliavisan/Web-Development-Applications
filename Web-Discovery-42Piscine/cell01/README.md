# Cell 01: First Steps into Web Development

## EN
**Project Overview**
This cell marks the transition from shell scripting to web technologies. The exercises are designed to teach the core syntax of HTML5 and CSS3, starting from basic document structure and moving towards multimedia integration, advanced styling, and responsive web design principles.

**Directory Content**
Based on the exercise requirements (Cell 1-0 to 1-6), this directory contains:
* **ex00 (Hello World)**: A fundamental `helloworld.html` file that displays "Hello World" in the browser to test environment setup.
* **ex01 (Title)**: Implementation of `piscineweb.html` featuring a large, centered main title using appropriate HTML tags.
* **ex02 (Image)**: Updates to the web page to include the integration of external image assets found via search engines.
* **ex03 (Link)**: Multimedia enhancement where an image is wrapped in an anchor tag to function as a clickable link to a specific URL.
* **ex04 (Basics or home)**: A comprehensive page using a specific pink background, a white centered title with special characters, side-by-side images with links, and a right-aligned footer with copyright info in monospace font.
* **ex05 (Tab)**: Introduction to data presentation using HTML tables, featuring cells with custom background colors and text.
* **ex06 (Responsive)**: Advanced CSS application where the previous table structure is modified to be fully responsive, ensuring it shrinks and covers the entire page during browser resizing.

**Technical Skills Demonstrated**
* **HTML5 Structure**: Writing semantic markup and understanding standard web headers.
* **CSS3 Styling**: Managing background colors, text alignment, and typography properties (italic, monospace).
* **Multimedia & Links**: Embedding images and implementing hyperlinking for navigation.
* **Table Layouts**: Organizing content using rows and cells with specific visual attributes.
* **Responsive Design**: Using CSS (without JavaScript) to create fluid layouts that adapt to various screen dimensions.

## RO
**Prezentare Proiect**
Această celulă marchează tranziția de la scripting shell la tehnologii web. Exercițiile sunt concepute pentru a preda sintaxa de bază a HTML5 și CSS3, începând de la structura de bază a documentului și avansând către integrarea multimedia, stilizarea avansată și principiile de design web responsiv.

**Conținutul Directorului**
Conform cerințelor exercițiilor (Cell 1-0 până la 1-6), acest director conține:
* **ex00 (Hello World)**: Un fișier fundamental `helloworld.html` care afișează "Hello World" în browser pentru a testa configurarea mediului.
* **ex01 (Title)**: Implementarea `piscineweb.html` care include un titlu principal mare și centrat folosind tag-uri HTML adecvate.
* **ex02 (Image)**: Actualizarea paginii web pentru a include integrarea resurselor de imagini externe.
* **ex03 (Link)**: Îmbunătățire multimedia în care o imagine este încadrată într-un tag ancoră pentru a funcționa ca un link către un URL specific.
* **ex04 (Basics or home)**: O pagină complexă care utilizează un fundal roz specific, un titlu alb centrat cu caractere speciale, imagini alăturate cu link-uri și un subsol aliniat la dreapta cu informații de copyright în font monospace.
* **ex05 (Tab)**: Introducere în prezentarea datelor folosind tabele HTML, incluzând celule cu culori de fundal și text personalizat.
* **ex06 (Responsive)**: Aplicare avansată CSS în care structura tabelului anterior este modificată pentru a fi complet responsivă, asigurându-se că se restrânge și acoperă întreaga pagină în timpul redimensionării browserului.

**Competențe Tehnice Demonstrate**
* **Structură HTML5**: Scrierea marcajului semantic și înțelegerea headerelor web standard.
* **Stilizare CSS3**: Gestionarea culorilor de fundal, alinierea textului și proprietățile de tipografie (italic, monospace).
* **Multimedia și Link-uri**: Inserarea imaginilor și implementarea hyperlink-urilor pentru navigare.
* **Layout-uri Tabelare**: Organizarea conținutului folosind rânduri și celule cu atribute vizuale specifice.
* **Design Responsiv**: Utilizarea CSS (fără JavaScript) pentru a crea layout-uri fluide care se adaptează la diverse dimensiuni ale ecranului.
