# Cell 00: Shell Discovery and Fundamentals

## EN
**Project Overview**
This initial cell serves as an introduction to the command line interface and the Unix filesystem. The exercises focus on navigating the terminal, automating basic tasks through shell scripting, and understanding how to manipulate files and directories using standard commands.

**Directory Content**
Based on the exercise requirements, this directory is organized into the following modules:
* **ex01 (Z)**: A simple file named `z` that returns the character "Z" followed by a newline when accessed via the `cat` command.
* **ex02 (midLS)**: A command line placed in a `midLS` file that lists all files and directories in the current path (excluding hidden files), separated by commas.
* **ex03 (count_files)**: A shell script (`count_files.sh`) that counts and displays the total number of regular files and directories within the current folder.
* **ex04 (argv)**: A shell program (`argv.sh`) designed to display up to three arguments passed to it during execution.
* **ex05 (build)**: A bash script (`build.sh`) that automates folder creation by prepending "ex" to the beginning of each argument name provided.



**Technical Skills Demonstrated**
* **Terminal Navigation**: Mastering the first commands to navigate, modify, and create within the filesystem.
* **Shell Scripting**: Writing basic programs using /bin/bash to automate terminal tasks.
* **Argument Handling**: Learning how to capture and display input parameters (argv) passed to a script.
* **I/O Redirection**: Using commands like cat and pipes to manage data output and display information.

## RO
**Prezentare Proiect**
Această primă celulă servește ca introducere în interfața liniei de comandă și în sistemul de fișiere Unix. Exercițiile se concentrează pe navigarea în terminal, automatizarea sarcinilor de bază prin scripturi shell și înțelegerea modului de manipulare a fișierelor și directoarelor folosind comenzi standard.

**Conținutul Directorului**
Conform cerințelor exercițiilor, acest director este organizat în următoarele module:
* **ex01 (Z)**: Un fișier simplu numit `z` care returnează caracterul "Z" urmat de o linie nouă atunci când este accesat prin comanda `cat`.
* **ex02 (midLS)**: O linie de comandă plasată într-un fișier `midLS` care listează toate fișierele și directoarele non-ascunse din calea curentă, separate prin virgulă.
* **ex03 (count_files)**: Un script shell (`count_files.sh`) care numără și afișează numărul total de fișiere obișnuite și directoare din folderul curent.
* **ex04 (argv)**: Un program shell (`argv.sh`) conceput pentru a afișa maximum trei argumente transmise acestuia în timpul execuției.
* **ex05 (build)**: Un script bash (`build.sh`) care automatizează crearea de foldere prin adăugarea prefixului "ex" la începutul numelui fiecărui argument furnizat.

**Competențe Tehnice Demonstrate**
* **Navigare în Terminal**: Stăpânirea primelor comenzi pentru a naviga, modifica și crea elemente în sistemul de fișiere.
* **Scripting Shell**: Scrierea de programe de bază folosind /bin/bash pentru a automatiza sarcini în terminal.
* **Gestionarea Argumentelor**: Învățarea modului de capturare și afișare a parametrilor de intrare (argv) transmiși unui script.
* **Redirecționare I/O**: Utilizarea comenzilor precum cat și a pipe-urilor pentru a gestiona ieșirea datelor și afișarea informațiilor.
