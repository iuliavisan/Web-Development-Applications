# Lab 7: Interactive Feedback Widget with DOM Manipulation

## EN
**Project Overview**
This laboratory focuses on creating a complex interactive component: a Star Rating Feedback Widget. The project integrates advanced DOM manipulation, event handling, and synchronized CSS styling to provide a modern user interface for collecting ratings and comments.

**Directory Content**
Based on the laboratory requirements and the specific files in this folder, the directory includes:
* index.html: Defines the widget structure, including the descriptive title, the five-star rating container (using images), a comment textarea, and control buttons (Submit and Reset).
* style.css: Manages the visual appearance of the widget, specifically defining the dimensions of the stars and the clear distinction between inactive (gray) and active (gold) states.
* script.js: Contains the core logic for the interactive rating system, including functions to toggle star states, track the currently selected rating, and handle the feedback submission process.

**Technical Skills Demonstrated**
* Complex Event Handling: Implementing click events that trigger visual changes across multiple document nodes simultaneously (lighting up all stars up to the selected one).
* Dynamic UI Feedback: Providing real-time visual confirmation of the user's selection from a 1 to 5-star scale.
* Data Collection and Display: Retrieving information from both the rating system and text input fields to generate a final confirmation message upon submission.
* State Management: Implementing a reset function that programmatically restores the document to its initial state (rating at 0 and all stars inactive).

## RO
**Prezentare Proiect**
Acest laborator se concentrează pe crearea unei componente interactive complexe: un Widget de Feedback cu sistem de notare prin stele. Proiectul integrează manipularea avansată a DOM-ului, gestionarea evenimentelor și stilizarea CSS sincronizată pentru a oferi o interfață modernă de colectare a evaluărilor și comentariilor.

**Conținutul Directorului**
Conform cerințelor de laborator și fișierelor specifice din acest folder, directorul include:
* index.html: Definește structura widget-ului, incluzând titlul descriptiv, containerul pentru cele cinci stele (folosind imagini), o zonă de text pentru comentarii și butoanele de control (Trimitere și Resetare).
* style.css: Gestionează aspectul vizual al widget-ului, definind în special dimensiunile stelelor și distincția clară între stările inactivă (gri) și activă (auriu).
* script.js: Conține logica de bază pentru sistemul interactiv de rating, incluzând funcții pentru comutarea stării stelelor, urmărirea ratingului selectat și gestionarea procesului de trimitere a feedback-ului.

**Competențe Tehnice Demonstrate**
* Gestionarea Complexă a Evenimentelor: Implementarea evenimentelor de tip click care declanșează schimbări vizuale simultane în mai multe noduri ale documentului (iluminarea tuturor stelelor până la cea selectată).
* Feedback UI Dinamic: Oferirea unei confirmări vizuale în timp real a selecției utilizatorului pe o scară de la 1 la 5 stele.
* Colectarea și Afișarea Datelor: Preluarea informațiilor atât din sistemul de rating, cât și din câmpurile de text pentru a genera un mesaj de confirmare final la trimiterea feedback-ului.
* Gestionarea Stării (State Management): Implementarea unei funcții de resetare care readuce programatic documentul la starea inițială (rating 0 și toate stelele inactive).
