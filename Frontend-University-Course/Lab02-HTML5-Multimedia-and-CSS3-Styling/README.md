# Lab 2: HTML5 Multimedia and CSS3 Visual Styling

## EN
**Project Overview**
This second laboratory introduces modern web standards, focusing on the integration of native multimedia elements and the fundamentals of visual design using CSS3. The exercises cover the transition from basic HTML to styled, interactive web interfaces.

**Directory Content**
Based on the laboratory requirements and files, this directory includes:
* multimedia.html: Demonstrates the native integration of audio and video files using HTML5 tags (<audio> and <video>) with support for multiple formats like MP3, Wav, Ogg, and MP4.
* stil.html: Focuses on CSS3 styling techniques, including the use of external and internal stylesheets to control page layout and typography.
* fonturi.html: Illustrates the @font-face rule, allowing the use of custom, non-standard web fonts to enhance document aesthetics.
* opacitate.html: Showcases visual effects such as image transparency and "mouse-over" hover effects using the CSS opacity property and filters.
* selectori.html: An introduction to CSS selectors for targeting specific HTML elements, classes, and IDs for precise styling.
* fisiere/: A resource folder containing the multimedia assets (relax.mp3, sunet.wav, bunny.ogg, video.mp4) and custom font files (.ttf) used in the exercises.

**Technical Skills Demonstrated**
* Multimedia Integration: Using HTML5 controls for audio/video playback without third-party plugins.
* Advanced Typography: Implementing custom fonts and controlling text properties through CSS.
* Visual Effects: Creating interactive UI elements using transparency, transitions, and hover states.
* CSS Architecture: Understanding the hierarchy of selectors and the application of styles across different HTML structures.

## RO
**Prezentare Proiect**
Acest al doilea laborator introduce standardele web moderne, concentrându-se pe integrarea elementelor multimedia native și pe fundamentele designului vizual utilizând CSS3. Exercițiile acoperă tranziția de la HTML de bază la interfețe web stilizate și interactive.

**Conținutul Directorului**
Conform cerințelor din laborator și fișierelor existente, acest director include:
* multimedia.html: Demonstrează integrarea nativă a fișierelor audio și video folosind tag-urile HTML5 (<audio> și <video>), oferind suport pentru formate multiple precum MP3, Wav, Ogg și MP4.
* stil.html: Se concentrează pe tehnici de stilizare CSS3, inclusiv utilizarea foilor de stil externe și interne pentru a controla layout-ul paginii și tipografia.
* fonturi.html: Ilustrează regula @font-face, permițând utilizarea fonturilor personalizate (non-standard) pentru a îmbunătăți estetica documentului.
* opacitate.html: Prezintă efecte vizuale precum transparența imaginilor și efecte de tip "mouse-over" (hover) folosind proprietatea opacity și filtre CSS.
* selectori.html: O introducere în selectorii CSS pentru țintirea elementelor HTML specifice, claselor și ID-urilor pentru o stilizare precisă.
* fisiere/: Un folder de resurse care conține fișierele multimedia (relax.mp3, sunet.wav, bunny.ogg, video.mp4) și fișierele de fonturi (.ttf) utilizate în exerciții.

**Competențe Tehnice Demonstrate**
* Integrare Multimedia: Utilizarea controalelor HTML5 pentru redarea audio/video fără plug-in-uri externe.
* Tipografie Avansată: Implementarea fonturilor personalizate și controlul proprietăților textului prin CSS.
* Efecte Vizuale: Crearea elementelor de UI interactive folosind transparența, tranzițiile și stările de hover.
* Arhitectură CSS: Înțelegerea ierarhiei selectorilor și aplicarea stilurilor pe diferite structuri HTML.
