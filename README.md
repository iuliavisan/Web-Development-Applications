# Web Discovery Piscine - Technical Portfolio

## EN
**Project Description**
This repository documents my intensive journey through the fundamentals of modern computing and web development. The "Discovery Piscine" is a hands-on learning experience focused on problem-solving, peer-learning, and the assembly of technical "bricks" to build functional software solutions.

**Curriculum Journey**
The program is divided into specialized cells, each introducing a new layer of technology:

* **Cell 00: Shell & System Fundamentals**
    * Focus: Mastering the Unix terminal and command-line interface.
    * Key Concepts: Filesystem navigation, I/O redirection, and automation via Shell Scripting (BASH).
* **Cell 01: HTML5 & CSS3 Basics**
    * Focus: Transitioning to web structure and visual presentation.
    * Key Concepts: Semantic HTML, multimedia integration, and the introduction of Responsive Design to ensure layouts adapt to all screen sizes.
* **Cell 02: Advanced Layouts & UI Components**
    * Focus: Mastering professional design patterns and CSS architecture.
    * Key Concepts: **Parallax Scrolling** effects for depth, **CSS-only Dropdown Menus** (logic without scripts), and structured UI components like User Profile Cards.
* **Cell 03: JavaScript & Dynamic DOM**
    * Focus: Adding behavior and interactivity to the web.
    * Key Concepts: Event handling, **DOM Manipulation**, data persistence using **Cookies**, and introduction to the **jQuery** library for efficient scripting.
* **Rush: Final Collaborative Project**
    * Focus: Integration and teamwork.
    * Key Concepts: Building a complete, cohesive website by combining all previously learned "bricks" into a unified product.

**Technical Stack**
* **Shell**: BASH.
* **Web**: HTML5, CSS3, JavaScript (Vanilla & jQuery).
* **Design**: Responsive Web Design, Parallax, UI/UX Components.
* **Version Control**: Git & GitHub.

## RO
**Descrierea Proiectului**
Acest repository documentează parcursul meu intensiv prin fundamentele informaticii și dezvoltării web moderne. "Discovery Piscine" este o experiență de învățare practică axată pe rezolvarea problemelor, învățarea între egali (peer-learning) și asamblarea "cărămizilor" tehnice pentru a construi soluții software funcționale.

**Parcursul Curriculei**
Programul este împărțit în celule specializate, fiecare introducând un nou nivel de tehnologie:

* **Cell 00: Fundamente Shell și Sistem**
    * Focus: Stăpânirea terminalului Unix și a interfeței în linie de comandă.
    * Concepte Cheie: Navigarea în sistemul de fișiere, redirecționarea I/O și automatizarea prin Shell Scripting (BASH).
* **Cell 01: Bazele HTML5 și CSS3**
    * Focus: Tranziția către structura web și prezentarea vizuală.
    * Concepte Cheie: HTML semantic, integrare multimedia și introducerea Designului Responsiv pentru a asigura adaptarea layout-urilor pe orice ecran.
* **Cell 02: Layout-uri Avansate și Componente UI**
    * Focus: Stăpânirea modelelor de design profesional și a arhitecturii CSS.
    * Concepte Cheie: Efecte de **Parallax Scrolling** pentru profunzime, **Meniuri Dropdown exclusiv CSS** (logică fără scripturi) și componente UI structurate (Carduri de Profil).
* **Cell 03: JavaScript și DOM Dinamic**
    * Focus: Adăugarea comportamentului și interactivității în web.
    * Concepte Cheie: Gestionarea evenimentelor, **Manipularea DOM**, persistența datelor folosind **Cookie-uri** și introducerea bibliotecii **jQuery**.
* **Rush: Proiectul Colaborativ Final**
    * Focus: Integrare și lucru în echipă.
    * Concepte Cheie: Construirea unui site web complet și coeziv prin combinarea tuturor tehnologiilor învățate anterior într-un produs unificat.

**Tehnologii Utilizate**
* **Shell**: BASH.
* **Web**: HTML5, CSS3, JavaScript (Vanilla & jQuery).
* **Design**: Design Web Responsiv, Parallax, Componente UI/UX.
* **Controlul Versiunii**: Git & GitHub.
