# Lab 4: Text Styling, Backgrounds, and CSS Box Model

## EN
**Project Overview**
This laboratory focuses on the detailed visual refinement of web elements. It covers advanced text styling, comprehensive background manipulation, the pseudo-classes for hyperlinking, and a deep dive into the CSS Box Model for precise layout control.

**Directory Content**
Based on the laboratory exercises and folder structure, this directory contains:
* exercitiu1.html: Focuses on text styling properties including horizontal and vertical alignment, text decoration (overline, underline, line-through), and text transformation (uppercase/lowercase).
* exercitiul2.html: Demonstrates font management, exploring font families, styles (italic, oblique), and font sizes using both absolute (px) and relative (em) units.
* exercitiul3.html: Implementation of background properties such as background-color, background-image insertion, and controlling repetitions (repeat-x, repeat-y, no-repeat) and positioning.
* exercitiul4.html: Advanced styling of navigation menus using hyperlinking pseudo-classes (link, visited, hover, active) combined with list management and floating elements.
* exercitiul5.html: Practical implementation of the CSS Box Model, illustrating the interaction between content, padding, border, and margin to calculate total element dimensions.
* image.jpg: A sample graphic resource used for background and content styling exercises.

**Technical Skills Demonstrated**
* Typography Control: Precise management of letter spacing, word spacing, and text indentation for improved readability.
* Dynamic Navigation: Creating professional navigation bars with interactive hover effects using CSS pseudo-classes.
* Box Model Architecture: Understanding how to calculate and manipulate the physical space an element occupies on a page.
* Background Management: Positioning and scaling background images to create layered visual designs without affecting document flow.

## RO
**Prezentare Proiect**
Acest laborator se concentrează pe rafinarea vizuală detaliată a elementelor web. Acesta acoperă stilizarea avansată a textului, manipularea complexă a fundalurilor, utilizarea pseudo-claselor pentru hyper-link-uri și o analiză aprofundată a Modelului Box CSS pentru controlul precis al machetării.

**Conținutul Directorului**
Conform exercițiilor din laborator și structurii folderului, acest director conține:
* exercitiu1.html: Se concentrează pe proprietățile de stilizare a textului, inclusiv alinierea orizontală și verticală, decorarea textului (tăiere, subliniere) și transformarea textului (majuscule/litere mici).
* exercitiul2.html: Demonstrează gestionarea fonturilor, explorând familiile de fonturi, stilurile (italic, oblic) și dimensiunile acestora folosind atât unități absolute (px), cât și relative (em).
* exercitiul3.html: Implementarea proprietăților de fundal, precum culoarea de fundal, inserarea imaginilor și controlul repetițiilor (repeat-x, repeat-y, no-repeat) sau a poziționării.
* exercitiul4.html: Stilizarea avansată a meniurilor de navigare folosind pseudo-clasele pentru link-uri (link, visited, hover, active), combinate cu gestionarea listelor și a elementelor flotante.
* exercitiul5.html: Implementarea practică a Modelului Box CSS, ilustrând interacțiunea dintre conținut, padding, bordură și margine pentru calcularea dimensiunilor totale ale elementelor.
* image.jpg: O resursă grafică utilizată pentru exercițiile de stilizare a fundalului și a conținutului.

**Competențe Tehnice Demonstrate**
* Controlul Tipografiei: Gestionarea precisă a distanței între litere/cuvinte și a indentării textului pentru o lizibilitate sporită.
* Navigare Dinamică: Crearea barelor de navigare profesionale cu efecte interactive la trecerea mouse-ului folosind pseudo-clase CSS.
* Arhitectura Modelului Box: Înțelegerea modului de calcul și manipulare a spațiului fizic pe care un element îl ocupă în pagină.
* Managementul Fundalurilor: Poziționarea și scalarea imaginilor de fundal pentru a crea design-uri vizuale complexe fără a afecta fluxul documentului.
